# Installation
```
composer install
```
## Update DB's config at .env
```
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=?
DB_USERNAME=?
DB_PASSWORD=?
```
## run migration
```
php artisan migrate
```

## run the app
```
php artisan serve
```

register an account then you will be able to add your favourite tv show.
